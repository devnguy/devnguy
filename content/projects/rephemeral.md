## Details

Rephemeral is a reference image viewing and practice tool for visual artists.
Users can choose to utilize their existing Pinterest boards, or their own local
images.

Random images will be chosen and displayed from the selected pool of images for
a set interval. Or displayed at intervals to simulate a life drawing session.

## Features

- Pinterest integration
- Local image integration
- Drawing session customization
